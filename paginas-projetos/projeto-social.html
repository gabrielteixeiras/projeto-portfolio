O que é este projeto?

Este projeto consiste em um site responsivo focado 100% no Front-end.

O site em questão reúne todas as redes sociais do professor e criador de conteúdo Gustavo Guanabara. Por meio dele, é possível selecionar nas opções disponíveis ao lado direito da tela qual rede social deseja visualizar. Ao clicar na rede social desejada, ela é exibida na tela do smartphone. Ao final, há a opção de clicar no botão "Acessar" para ser redirecionado à rede social escolhida.

O que foi usado neste projeto?

Neste projeto, foi utilizado HTML e CSS. O foco principal do projeto está voltado para o aprendizado sobre o elemento HTML iframe, o qual é utilizado para exibir as redes sociais na tela do smartphone.

Este projeto foi desenvolvido com base no capítulo 23 do curso "Curso HTML5 e CSS3: módulo 4 de 5" da plataforma Curso em Vídeo.
