O que é este projeto?

Este projeto consiste em um site responsivo focado 100% no Front-end.

O AluraBooks é um site dedicado à venda de livros técnicos do Grupo Alura. Através dele, os usuários podem acessar um menu hambúrguer localizado no cabeçalho da página, explorar os livros em carrosséis, visualizar cards distribuídos pela página, além de encontrar uma variedade de botões, dentre outras funcionalidades.

O que foi usado neste projeto?

Neste projeto, foi utilizado HTML, CSS e JavaScript. O foco principal está direcionado para o aprendizado sobre responsividade com a abordagem Mobile-first, a qual é amplamente utilizada e recomendada no desenvolvimento web.

Este projeto foi desenvolvido com base no curso "HTML e CSS: responsividade com mobile-first" da plataforma Alura.
