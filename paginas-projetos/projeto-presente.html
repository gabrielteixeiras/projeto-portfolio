O que é este projeto?

Este projeto consiste em um site responsivo focado 100% no Front-end.

Este projeto foi desenvolvido com o objetivo de proporcionar alegria aos amigos e familiares que celebram aniversários. Nele, é possível visualizar uma cartinha contendo palavras de carinho, além de possuir suas fotos preferidas.

O que foi usado neste projeto?

Neste projeto, foi utilizado HTML e CSS. O foco principal está direcionado para o aprendizado sobre grid, flexbox, entre outros conceitos importantes de layout, os quais foram empregados para posicionar os elementos presentes no projeto de forma eficiente e organizada.

Este projeto foi desenvolvido com base na inspiração de um layout disponibilizado pela plataforma Alura.
